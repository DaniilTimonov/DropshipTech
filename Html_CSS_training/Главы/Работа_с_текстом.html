Прописные и строчные символы

работаем с свойством text-transform

при значении text-transform: uppercase; для всех символов в тексте используется верхний регистр

при значении text-transform:lowercase; для всех символов в тексте используется нижний регистр

при значении text-transform:capitalize; все символы будут начинатся с прописной буквы
