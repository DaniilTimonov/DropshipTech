Перекрывающиейся элементы z-index

Если вы хотите задать, какой элемент будет находится на переднем плане, вы можете использовать z-index. Его значением является число, чем выше число. тем ближе элемент будет к переднему плану

Например. Голова персонажа имеет значение 10, а картинка балончика 5. Персонаж перекрывает собой картинку с балончиком. Поменяв обратно значения все будет наоборот, то значение по z-index, что будет выше будет являться доминирующим
